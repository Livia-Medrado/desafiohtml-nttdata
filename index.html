Módulo 1 - Primeiros Passos com HTML1

Neste módulo, foram abordados temas importantes para o primeiro contato com HTML.

É de extrema importância adquirir uma boa base de conhecimento quando se inicia uma nova jornada de aprendizado. Conhecer mais sobre as origens, principais nomes da história da computação e como se deu a sua evolução são pontos importantes para que possamos começar a nos integrar ao mundo da tecnologia. Através do HTML já conseguimos dar os primeiros passos, desenvolvendo uma página web, como esta! Para isso, é necessario aprender a estrutura básica, tags, atributos e etc. Nos tópicos abaixo estão os principais assuntos deste primeiro módulo, que serão a base de um bom aprendizado, e uma sintese sobre cada um deles. Aproveite!

Principais Tópicos:

  1. História dos Computadores
  2. Client X Server
  3. Estrutura Básica do HTML
