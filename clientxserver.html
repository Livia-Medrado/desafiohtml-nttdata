Client X Server

O que é Client?

É a parte que consome a informação. Dispositivos que vão receber as informações através da internet. Ex.:Notebooks, celulares, pcs...
Essas informações estão hospedadas no servidor, que processa essas informações e retorna arquivos estáticos(HTML, CSS, JS, imagens, áudios, etc...) para o client. Esses arquivos são interpretados pelos navegadores.

O que é Server?

São os servidores. As partes que enviam as informações para o client. São computadores robustos, centrais, que armazena os arquivos, processam e os formanecem ao client. Existem também os servidores de segurança, de streaming, de e-mail e web.
